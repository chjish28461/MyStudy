    • JavaScript 变量
    JavaScript 变量是存储数据值的容器。
    在本例中，x、y 和 z 是变量：
    实例
    var x = 7;
    var y = 8;
    var z = x + y;
    从上例中，您可获得：
    x 存储值 7
    y 存储值 8
    z 存储值 15
    类似代数
    在本例中，price1、price2 以及 total 是变量：

    实例
    var price1 = 7;
    var price2 = 8;
    var price2 = 12;
    var total = price1 + price2 + price3;
